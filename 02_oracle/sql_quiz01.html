1. Salary가 15,000 이상의 직원의 Id를 오름차순으로 배열하시오.

Answer select salary, employee_id from employees where salary >= 15000 order by employee_id asc; SALARY EMPLOYEE_ID ---------- ----------- 24000 100 17000 101 17000 102

2. Manger_ID 가 100 이며, 01 년도에서 02 년도 사이에 입사한 직원의 Salary 가 5% 인상이 된다. 이들의 현재 Salary와 인상된 Salary, Employee_id 그리고 Manager_ID 를 출력하시오.

Answer select manager_id, employee_id, salary, salary*1.05 as Increased from employees where manager_id = 100and hire_date between '01/01/01'and'02/12/31'; EMPLOYEE_ID MANAGER_ID SALARY INCREASED ----------- ---------- ---------- ---------- 102 100 17000 17850 114 100 11000 11550

3. Employee_ID가 100 사이의 110인 직원 중 First_name의 문자열이 짝수인 직원인 월급을 반올림하고 직원의 ID 와 반올림된 월급을 출력하라

Answer select employee_Id, round(salary) from employees where mod(length(First_name),2) =0 and employee_idbetween100 and110; EMPLOYEE_ID ROUND(SALARY) ----------- ------------- 100 24000 109 9000 110 8200
